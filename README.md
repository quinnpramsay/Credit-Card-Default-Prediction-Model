# Predicting Credit Card Default

This project predicts whether a credit card client will default on their next payment using machine learning. The dataset has 30,000 records from the UCI Machine Learning Repository.

## Models

Two models were trained and compared:
- Decision Tree (82% accuracy, best for fewer false alarms)
- Naive Bayes (70% accuracy, best for catching more defaulters)

## Tools

Python, pandas, NumPy, scikit-learn, matplotlib, seaborn

## How to Run

1. Clone the repo
2. Download the dataset: https://archive.ics.uci.edu/dataset/350/default+of+credit+card+clients
3. Put the dataset file in the project folder
4. Open the notebook and run all cells
